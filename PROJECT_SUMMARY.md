## 🎯 Основные компоненты

### 1. **Core Agent** (`src/core/agent.py`)
- Основной класс для управления диалогом
- Поддержка ролей: пациент, врач, интервьюер
- Управление сеансами и сбор анамнеза

### 2. **LLM Manager** (`src/core/llm_manager.py`)
- Интеграция с Ollama (локальная LLM)
- Генерация ответов на русском/украинском
- Извлечение медицинской информации

### 3. **Voice Components** (`src/voice/`)
- **SpeechRecognizer**: Распознавание речи (Whisper)
- **TextToSpeech**: Синтез речи (Piper)
- Поддержка непрерывного распознавания

### 4. **Medical Logic** (`src/core/medical_logic.py`)
- База симптомов → диагнозы
- Предложение возможных диагнозов
- Информация о болезнях и лечении

### 5. **RAG System** (`src/medical/rag.py`)
- Retrieval-Augmented Generation
- Поиск в медицинской базе знаний
- Контекстное дополнение ответов LLM

### 6. **Questionnaire** (`src/medical/questionnaire.py`)
- Структурированный сбор анамнеза
- Валидация ответов
- Экспорт в JSON

### 7. **User Interfaces**
- **CLI** (`src/ui/cli.py`): Интерактивная командная строка с цветным выводом
- **REST API** (`src/ui/api.py`): FastAPI с WebSocket поддержкой

### 8. **Memory Management** (`src/core/memory.py`)
- Управление историей разговора
- Контекстное окно для LLM
- Экспорт/импорт сеансов

## 🚀 Режимы работы

### Командная строка (CLI)
```bash
python main.py --mode cli [--voice] [--debug]
```
- Интерактивный диалог с агентом
- Поддержка команд (`/help`, `/doctor`, `/patient` и т.д.)
- Опциональный голосовой режим

### REST API
```bash
python main.py --mode api [--host 0.0.0.0] [--port 8000]
```
- HTTP endpoints для интеграции
- WebSocket для реального времени
- Автоматическая документация (Swagger)

### Docker
```bash
docker-compose up
```
- Полная контейнеризация
- Ollama + приложение в одном составе
